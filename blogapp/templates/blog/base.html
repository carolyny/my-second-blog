Pushing Code

Hello, I'm Carolyn. In May 2015, I took part in a Django Girls workshop. I was amazed at what I managed to complete in one day and how much I enjoyed coding. This website, a work in progress, is the second version of the blog I created on that day (I like pictures so decided to add one to each blog entry. It took me ages to get it to work but it felt so good when it did!).

I aim to write about my coding journey. It's started slowly but if I want to change career (which I do), I need to learn, to practise, to write code and regularly push code. So introduction over, time to get properly started.
